En el código fuente de la página web los caracteres Latin-1 suelen mostrarse directamente, pero el resto aparece codificado con el formato &nombre; o bien &codigo_decimal; o bien &xcodigo_hexadecimal; es decir, primero el símbolo "&" (en inglés este símbolo se llama ampersand, en español se llama et), después el nombre del carácter o el código numerico (si es hexadecimal precedido de una letra "x"), y por último un punto y coma ";". Por ejemplo, si una página web contiene la letra griega alfa (α), en el código fuente se puede encontrar una de estas tres expresiones equivalentes: &alpha; &#945; &#x3b1;
